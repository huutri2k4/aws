### Week 3 Objectives:
* Learn how to set up cost alerts (AWS Budgets) to prevent unexpected charges while doing labs.
* Understand how to create an Amazon EC2 instance and connect to it from a local machine.

| Day | Task | Start Date | End Date | References |

### Week 3 Outcomes:
* Successfully configured AWS Budgets to monitor costs and prevent unexpected bill shocks during labs.
* Know how to launch a complete EC2 instance by selecting the OS, instance type, and storage size (EBS).
* Got a grasp of Security Groups for opening and closing network ports.
* Capable of using the local terminal to SSH into a remote cloud server and managing its lifecycle (stopping it when not in use).
